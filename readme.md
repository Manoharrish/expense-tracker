# Expense Tracker

A simple Expense Tracker web application built using PHP and MySQL.

## Features
- Add expenses
- View all expenses
- Delete expenses
- Filter expenses by category
- Automatic total calculation

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

## Database Structure
Table: expenses
- id (INT, Primary Key)
- title (VARCHAR)
- amount (DECIMAL)
- date (DATE)
- category (VARCHAR)

## How to Run the Project
1. Install XAMPP
2. Start Apache and MySQL
3. Place project folder in `htdocs`
4. Create database `expense_tracker`
5. Create table `expenses`
6. Open `http://localhost/expense-tracker`
